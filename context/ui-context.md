# UI Context

## Theme
- **Dark Mode Priority:** Gyms are often dimly lit; dark mode reduces eye strain.
- **Color Palette:** High-energy accents (e.g., Electric Blue or Lime) for action buttons.

## Layouts
- **Workout Mode:** A focused, distraction-free view that shows "Last Week's Stats" as placeholders in the input fields.
- **Analytics:** Clean, responsive charts using `Recharts` or `Tremor`.
